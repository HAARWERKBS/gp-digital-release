# GP Digital - Gesellenprüfung Digital

Software zur digitalen Erfassung und Berechnung der Gesellenprüfung im Friseurhandwerk.

## Download

| Plattform | Download |
|-----------|----------|
| **macOS** | [GP Digital DMG](https://github.com/HAARWERKBS/gp-digital-release/releases/latest) |
| **Windows** | [GP Digital Setup](https://github.com/HAARWERKBS/gp-digital-release/releases/latest) |

Aktuelle und alle bisherigen Versionen finden Sie unter [Releases](https://github.com/HAARWERKBS/gp-digital-release/releases).

## Funktionen

- Verwaltung von Prüflingen und Jahrgängen
- GPO-konforme Notenberechnung (Teil 1 + Teil 2)
- Erfassung von Prüfer-Einzelbewertungen
- PDF-Export: Niederschriften, Zeugnisse, Gesellenbriefe
- Excel-Export mit Ergebnissen und Detail-Bewertungen
- Datensicherung und -wiederherstellung
